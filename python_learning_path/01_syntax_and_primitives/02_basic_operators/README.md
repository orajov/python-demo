# Základní operátory

Tato lekce představuje téma základní operátory a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- Aritmetika, //, %, **, porovnání
- == vs is
- Logické operátory

## Příklady
```python
print('Mocnina:', 2 ** 3, 'Modulo:', 10 % 3)
```

```python
print('Porovnání hodnot:', 3 == 3, 'Identita řetězců:', 'a' is 'a')
```

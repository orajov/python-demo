# Proměnné a přiřazení

Tato lekce představuje téma proměnné a přiřazení a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- Proměnné, typy, přiřazování, přepis hodnot
- Čitelné názvy proměnných
- Kontrola hodnot pomocí print nebo type

## Příklady
```python
name = 'Lenka'
age = 25
print('Jméno:', name, 'Věk:', age)
```

```python
age = age + 1
print('Aktualizovaný věk:', age)
```

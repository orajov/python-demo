# None a bool

Tato lekce představuje téma none a bool a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- None, pravdivostní hodnota objektů, bool(x)
- Rozlišení prázdných hodnot a False
- Používání v podmínkách

## Příklady
```python
value = None
print('Je hodnota nastavena?', value is None)
```

```python
items = []
print('Prázdný list je false?', bool(items))
```

# Slovníky a množiny

Tato lekce představuje téma slovníky a množiny a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- dict, set: klíče, hodnoty, iterace
- Typické operace přidání a odebrání
- Použití pro rychlé vyhledávání

## Příklady
```python
prices = {'jablko': 25, 'banán': 30}
print('Cena jablka', prices['jablko'])
```

```python
seen = set()
seen.add('uživatel')
print('Obsah množiny', seen)
```

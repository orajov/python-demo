# Sekvence a práce s nimi

Tato lekce představuje téma sekvence a práce s nimi a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- str, list, tuple: indexování, slicing
- Změnitelnost vs neměnitelnost
- Iterace přes sekvence

## Příklady
```python
text = 'Python'
print('První písmeno', text[0], 'Řez', text[1:4])
```

```python
numbers = [1, 2, 3]
numbers.append(4)
print('List je měnitelný:', numbers)
```

# Číselné typy

Tato lekce představuje téma číselné typy a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- int, float, převody, zaokrouhlování
- Matematické funkce v modulu math
- Bezpečná práce s číselnými vstupy

## Příklady
```python
value = 7
print('Jako float', float(value))
```

```python
import math
print('Zaokrouhlení', round(3.75), 'Druhá odmocnina', math.sqrt(16))
```

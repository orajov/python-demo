# Python Learning Path

Tento projekt obsahuje strukturovanou sadu lekcí pro samostudium Pythonu. Každá složka představuje jednu oblast a jednotlivé podkapitoly vždy nabízejí krátký úvod, přehled teorie, příklady a samostatné úkoly bez řešení.

Projděte lekce v pořadí složek, spusťte ukázky v souborech `examples.py` a doplňte implementace v `exercises.py`. Veškeré materiály jsou psané česky, identifikátory v kódu jsou anglicky podle konvence snake_case.

# Cyklus for a while

Tato lekce představuje téma cyklus for a while a nabízí krátké ukázky, které lze hned spustit. Obsah je stručný, aby bylo možné rychle pochopit hlavní principy a vyzkoušet si je v praxi.

## Teorie
- for přes range a kolekce
- while, vhodné použití
- Inicializace a podmínky ukončení

## Příklady
```python
for number in range(3):
    print('Opakování', number)
```

```python
names = ['Eva', 'Adam']
for name in names:
    print('Jméno', name)
```
